Windows下使用Hexo+Github搭建个人博客

准备工作

安装Node.js
下载地址：传送门
去 NodeJs 官网下载相应版本，进行安装即可。安装过程为默认下一步到底。
可以通过node -v的命令来测试NodeJS是否安装成功

安装Git
下载地址：传送门
去 Git 官网下载相应版本，进行安装即可。 安装过程为默认下一步到底。

准备Github账号
传送门
如果已经有帐号了那么接下来就是本地生成SSH KEY添加到Github
因为已经安装了Git 所以首先任意位置右键打开 Git Bash 运行如下命令

红线部分为你的Git账户邮箱
此步成功后在用户文件夹下会生成.ssh文件夹 进入以后会有id_rsa(私钥) 与 id_rsa.pub(公钥) 右键记事本打开 id_rsa.pub 复制里面全部内容 粘贴在GIT后台添加SSH KEY 处

搭建博客

配置github

登录Github，点击 “Start a project”，新建一个版本库
输入仓库名： 你的 Github名称.github.io 。然后点击创建
注意：创建名字，一定要用github的用户名，不然显示不出来，因为githubPage只能你的用户名。
接下来点击最后边的 settings 随意选择一个模版 启用
至此你离完成已经不远了 试着打开自己在github的静态网址http://你的帐号.github.io，你会发现，打开是你自己刚才选择静态站点模版。

恭喜你 这是通向成功的最后一步 将本地hexo项目托管到Github

打开hexo目录 修改配置文件_config.yml。

最后一步 安装hexo-deployer-git插件 把代码托管到Github上
